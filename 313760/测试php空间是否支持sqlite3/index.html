SQLite3 扩展自 PHP 5.3.0 起已默认启用。 允许在编译时使用 –without-sqlite3 禁用之。

Windows 用户必须启用 php_sqlite3.dll 方可使用该扩展。自 PHP 5.3.0 起，此扩展的 DLL 文件 已包含于 Windows 版的 PHP 发行包中。
