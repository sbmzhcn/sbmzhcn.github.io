另外，需要注意Javascript中函数的两种声明方式存在的坑：

function fn(){} //函数声明式
var fn = function(){}; //函数表达式
对于函数声明式，解析器会确保在所有代码执行之前声明已经被解析。而对于函数表达式，与定义其它基本类型变量一样，逐句执行并解析。
