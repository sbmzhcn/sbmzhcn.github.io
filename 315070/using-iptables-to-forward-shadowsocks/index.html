利用iptables实现shadowsocks中继/转发

发表于 2018-01-23 | 分类于 技术 | | 本文总阅读量 次 | 字数统计: 277

中继的原因是因为一些ip在我们本地的电脑上是无法连通的，但在阿里云等其它VPS上面就可以访问，这样我们可能通过中继来继续使用我们的shadowsocks.

开启系统的转发功能

vi /etc/sysctl.conf

将 net.ipv4.ip_forward=0 ，修改成 net.ipv4.ip_forward=1 。

编辑后使用以下命令让配置马上生效

sysctl -p
