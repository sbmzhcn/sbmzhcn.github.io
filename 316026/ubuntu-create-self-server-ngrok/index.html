发表于 2018-03-15 | 分类于 技术 | | 阅读次数: | 字数统计: 1,197

今天来郑州公司迁移 Google AdWords API 系统，想在上海访问这边的内网，这边是没有固定 IP的，这边就用到了内网穿透利器ngrok 了，这个东西微信开发也会用到。
