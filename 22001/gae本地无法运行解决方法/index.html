今天在本地运行micolog出现错误regex invalid: unbalanced parenthesis上网搜了下，终于在一篇英文里找到解决方法。

\dev_appserver.py from:

      regex = os.path.join(re.escape(regex), ‘(.*)’)
      path = os.path.join(path, ‘\1’)

to:

      regex = re.escape(regex) + ‘/(.*)’
      path = path + ‘/\1’
